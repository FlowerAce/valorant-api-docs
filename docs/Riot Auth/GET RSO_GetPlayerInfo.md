# GET RSO_GetPlayerInfo

Get the PUUID from a token  


Method: `GET`  
URL: `https://auth.riotgames.com/userinfo`  
Headers:
 - `Authorization`: `Bearer {Riot token}`

Variables:
 - `{Riot token}`: Read [Common Components - Riot Token](../common-components.md#riot-token)
